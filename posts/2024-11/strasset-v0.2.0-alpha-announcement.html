STRAsset v0.2.0 alpha 的更新公告

前文

本插件在此暫時更名為 STRAsset ，之後將會將前面的 STR 修改為新的工作室的名稱簡寫。

插件內部的資料夾也已經有了些修改，相同的功能或是類別的文件大部分都已經被移動到了相同的資料夾內。

之後一定還會有更多的檔案被歸類，如果有修改程式碼的使用者，在此還要說聲抱歉。


v0.2.0 alpha - 物件與特效 -

這次的 v0.2.0 alpha 的主要要素為 物件與特效 ，主要的內容是將角色腳本更改為物件腳本，並從中並從中衍生出角色腳本跟特效物件腳本這兩個分支。

如果在未來使用了 TRScript 創作了其他類型的腳本，就可以透過繼承 STRObject 來縮短工作流程與時間。

之後將會有關於兩個腳本的範本，以方便第一次使用的使用者。


v0.2.0 alpha 中的細微優化

在 v0.2.0 中，對於了一些不太順手的部分進行了優化

以下為在這次的更新中進行的優化：

  • 優化了腳本系統，將角色腳本更改為物件腳本，並從中衍生出角色腳本跟特效物件腳本
  • 優化了資料系統，將全數的資料以 Data Asset 的方式處理
  • 優化了判斷式，目前可以執行多層的判斷
  • 優化了 elseif ，將其簡短為 elif
  • 優化了碰撞箱的邏輯
  • 優化了輸入緩衝，目前還不確定會不會出現記憶體遺失等錯誤
  • 優化了狀態的執行邏輯
  • 優化了角色的招式鎖定
  • 優化了角色的二段跳，將會在第二段改變面向

v0.2.0 alpha 中會處理的修正

在進行 v0.2.0 的製作時，我也發現了一些之前沒有發現的錯誤與問題，並且進行了修正。

以下為在這次的更新中進行的修正：

  • 修正了角色在面相左側時(2P側)的2段跳方向錯誤
  • 修正了角色的行為結束時的狀態會變更為 "" 的問題

TRScript 的未來規劃

在製作此版本前，本來就已經有了一些構想，但是內容龐大，沒有時間也還沒有能力製作。

以下會列出之後有極大的機率會加入或是完善的功能：

  • 完善物件腳本的內容
  • 完善投擲的功能
  • 加入音效的功能
  • 加入角色語音的功能
  • 加入角色的眨眼功能
  • 加入遊戲系統的腳本
  • 加入2D的系統

Github 頁面的更動

雖然 Github 本身有一個 Wiki 的功能，但是我已經有一個網站了，所以我不想要花費多餘的時間去更新 Github 的頁面中的內容。

因此之後的 Github 頁面將會簡化為只剩下基礎的介紹。


結語

雖然之前將 STRAsset 發布了，但也是在發布的不久後發現了其中有很多沒有發現與修正的錯誤，所以在這幾天我都花了些時間進行了部分的修正。

然後目前也已經更新了 v0.2.0 alpha ，請前往 Github 頁面 取得更新。


到了這裡，這篇公告也差不多進入了尾聲。

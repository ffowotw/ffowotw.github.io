設置專案

本篇教學將會教導您該如何設置可以執行此插件的專案。

重要

雖然使用此插件不會需要C++的想關知識，但是 TRScript 為在C++的環境下編寫的插件，因此在建立新專案時請選擇使用C++。

將插件從 Github 複製到資料夾中

請打開到專案的位置，如果沒有Plugins的資料夾請創建該資料夾。

接著請準備將此插件從 Github 上面複製到資料夾中，點擊上方的目錄位置並輸入以下的命令開啟 cmd：

cmd

打開了 cmd 之後，在 cmd 輸入以下的命令將此插件複製到該資料夾內：

git clone https://github.com/ffowotw/STRAsset.git

注意

由於此插件不會自動更新，請在必需時查看是否有新的版本。

如有新的版本，請在 cmd 輸入此命令以更新插件：

git pull

在專案中啟用此插件

安裝好此插件之後就能在插件編輯介面中看到名為 STRAsset 的插件了，如果還沒有啟用，請啟用此插件並重新開啟專案。

注

如果您是在專案開啟的狀態下安裝了此插件，需要按下位於遊玩左側的編譯按鈕才能看到位於插件編輯介面中的插件了。
